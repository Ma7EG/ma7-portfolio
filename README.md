# Mahmoud Ali - JRPG  Portfolio

An interactive 8-bit JRPG-style portfolio for Mahmoud Ali, a Junior .NET Backend Developer. The application showcases my skills, experience, and projects through a retro game-like interface.

## Features

* **Active Quest Log:** Complete specific tasks (such as starting the game, collecting secret coins, upgrading stats, or chatting) to earn score and XP.
* **Character Attributes Sheet:** Use earned Skill Points (SP) to dynamically upgrade player stats.
* **Secret Coins:** Find and collect 5 hidden gold coins across the screen.
* **Retro Audio Synthesizer:** Synthesizes retro sound effects in real-time using the Web Audio API.
* **NPC AI Chatbot:** Talk to a custom AI Clone trained with my professional background, powered by Pollinations AI.

## Project Structure

* **Level 1 (About):** Class info, stats upgrade panel, and NPC dialogue console.
* **Level 2 (Experience):** Internships and education details.
* **Level 3 (Projects):** Grid showcasing key projects (Torostack, Podcasty, MedAura).
* **Level 4 (Achievements):** Problem-solving highlights and competitive programming profile links.

## Technologies Used

* **Frontend:** React, Tailwind CSS, Lucide icons, Motion
* **Backend:** Node.js, Express (proxying the AI Chat requests)
* **Audio:** Web Audio API (real-time synthesis)

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
